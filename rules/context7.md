# Context7 használat

- **Mi**: `context7` MCP szerver — friss, verziókövetett library/framework/SDK/CLI dokumentáció.
- **Használd**: külső library viselkedésével kapcsolatos kérdésnél (a tréning adat elavulhatott).
- **Ne használd**: saját business logika, általános programozási koncepciók, belső kódhiba.
- **Workflow**: `resolve-library-id` → `get-library-docs` / `query-docs`.
- **Preferencia**: library docs esetén Context7 > web search.
