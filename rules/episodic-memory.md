# Episodic memory

- **Mi**: `episodic-memory` plugin — szemantikus keresés a korábbi Claude Code beszélgetésekben.
- **Használd**: session elején, elakadáskor, vagy ha a user korábbi munkára hivatkozik.
- **Ne használd**: rutin feladatoknál, tisztán technikai kérdéseknél (arra `context7` való).
- **Workflow**: `mcp__plugin_episodic-memory__search` → `read` (ha releváns találat van).
- **Fontos**: mindig verifikáld a jelenlegi kód ellenében — a memória elavulhat.
